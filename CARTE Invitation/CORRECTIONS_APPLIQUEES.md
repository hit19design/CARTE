## 🔄 Fallbacks automatiques

Si une fonctionnalité n'est pas supportée :

### **Animations**
- Complexes → Simples
- CSS3 → CSS2
- Aucune → Version statique

### **Polices**
- Google Fonts → Web Safe
- Web Safe → Système
- Système → Arial/Times

### **JavaScript**
- Moderne → Classique
- Classique → Désactivé
- Désactivé → CSS seulement
